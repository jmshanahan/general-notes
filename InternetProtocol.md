# Internet Protocol

## External IP Address

Sample of commands to find an external IP Address


- method 1  
	curl ifconfig.me  
	
- method 2  
	curl ipinfo.io/ip/ 

- method 3  
	curl api.ipify.org

- method 4  
	curl chekip.dyndna.org

- method 5  
	You can determine it from a web browser ie whatsismyipaddress.com
	

## Internal IP Address

- method 1
	ip a, ip addr, The default is show ie ip addr show
